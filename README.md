# test-map

This project is a test task involving working with a cart and the Google API.

## Demo

Check out the live demo: [Demo Link](https://test-cart-task.vercel.app/)

## Technologies Used

- React
- TypeScript
- Firebase

## Install dependencies

npm install

## To run the project in development mode:

npm run dev
